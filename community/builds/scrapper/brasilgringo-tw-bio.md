## Context

Captured as a standalone Mids' Reborn forum-export block; no accompanying parent
guide was captured with it, so the design rationale is implicit. The pick set
suggests a soft-cap-defense build that layers Shadow Meld (Soul Mastery) for
emergency defense spikes, runs Tough/Weave + Maneuvers for the standing defense
layer, and uses Hasten + Mids `RechRdx-I` slotting in Hasten / Build Momentum to
chase perma-Hasten. Heavy hitters are slotted procmonster-style: Hecatomb +
Achilles' Heel proc on Rend Armor, Armageddon + FF +Recharge proc on Whirling
Smash, Superior Critical Strikes 6-piece on Arc of Destruction. Incarnate plan
is Musculature / Ion / Degenerative / Ageless / Assault.
