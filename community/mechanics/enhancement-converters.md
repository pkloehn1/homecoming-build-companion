## Summary

- Enhancement Converters are a special salvage type that let you swap one set IO (crafted or attuned) for another; they drop randomly from mobs, can be bought on the Auction House, or are purchased from Merit Vendors at a 3-converters-per-1-merit rate.
- There are three conversion modes: In-Set (3 converters, returns a different piece from the same set), Out-of-Set by Type (2 converters, returns a piece from a different set in the same enhancement category like Defense, PBAoE Damage, Healing, etc.), and Out-of-Set by Rarity (1 converter, returns any piece of the same rarity bucket — the cheapest mode and the workhorse for "converter roulette").
- The rarity buckets that conversions stay within are: Uncommon, Rare, Very Rare (purple), PvP, Attuned/ATO, Winter (WIO), and Universal Damage (currently only Overwhelming Force).
- Key limitations: conversions are level-locked, so you can't convert into a set that doesn't exist at the same level; you generally cannot upgrade rarity tiers (Uncommon to Rare conversion does not appear to happen via converters); and you can get stuck looping between two sets and waste a stack of converters.
- The basic monetization workflow (refined in materially-additive replies from Shinobu and Robotech_Master): craft cheap, low-demand recipes whose level band only has a small number of same-type sets, do a single Out-of-Set by Type conversion to force-funnel into a desired set, then run Out-of-Set by Rarity at 1 converter per try until you land on a high-value piece — exploiting the fact that regular and attuned versions of the same IO are fungible on the AH, so you don't need to attune separately.
